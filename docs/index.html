kmeans

src/kmeans.js

K-means algorithm

kmeans(data: Array<Array<number>>, K: number, options: object?): KMeansResult
Parameters
data (Array<Array<number>>) Points in the format to cluster [x,y,z,...]
K (number) Number of clusters
options (object?) Option object
Name Description
options.maxIterations number (default 100) Maximum of iterations allowed
options.tolerance number (default 1e-6) Error tolerance
options.withIterations boolean (default false) Store clusters and centroids for each iteration
options.distanceFunction function (default squaredDistance) Distance function to use between the points
options.seed number? Seed for random initialization.
options.initialization (string | Array<Array<number>>) (default 'kmeans++') K centers in format [x,y,z,...] or a method for initialize the data:
  • You can either specify your custom start centroids, or select one of the following initialization method:
  • 'kmeans++' will use the kmeans++ method as described by http://ilpubs.stanford.edu:8090/778/1/2006-13.pdf
  • 'random' will choose K random different values.
  • 'mostDistant' will choose the more distant points to a first random pick
Returns
KMeansResult: Cluster identifier for each data dot and centroids with the following fields:
  • 'clusters': Array of indexes for the clusters.
  • 'centroids': Array with the resulting centroids.
  • 'iterations': Number of iterations that took to converge
